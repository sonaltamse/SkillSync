# SkillSync

Lightweight project containing a small Express backend (in `backend/`) for demonstration and development.

## Quickstart

Requirements: Node.js (16+ recommended), npm

1. Install dependencies for the backend:

   cd backend
   npm install

2. Run in development mode (requires nodemon):

   npm run dev

3. Or run the server directly:

   npm start

The server listens on PORT (default 5000). Endpoints include `/`, `/json`, `/html`, and `/health`.
